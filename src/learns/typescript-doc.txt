*** 4 dac diem cua lap trinh huong doi tuong:
    encapsulation, abstraction, inheritance, and polymorphism

    - abstraction (TRUU TUONG):     
        + la viec xac dinh nhung thuoc tinh phuong thuc can thiet ke trong class
    - inheritance (KE THUA): 
        + lop nay co the ke thua tu lop kia cac phuong thuc cua lop kia
        + extend
    - polymorphism (DA HINH): 
        + viet lai cac thuoc tinh va phuong thuc cua lop khac
    - encapsulation (DONG GOI): 
        + tinh chat cho phep nguoi dung hay doi tuong khac thay doi dl cua dt hien Tai
        + acccess modifier
        + static

    Dinh nghia class: thuoc tinh, phuong thuoc (constructor, getter, setter, ...)

-Static trong lap trinh huong doi tuong: 
    Static la mot trong nhung cong cu giup nguoi lap trinh phan quyen truy cap:
        - mot gia tri trong class
        - mot phuong thuc trong class
        - Giup ta su dung thuoc tinh, phuong thuc cua mot class ma ko can khai bao mot instance moi

** Accessor trog typescript:
    access: truy cap 
    or: Nguoi
    - Chi dinh cach nguoi dung truy cap (doi tuong) truy cap khac giao tiep voi du lieu trong OOP
    - La cac ham de truy cap den doi tuong trong mot class
    - Tai sao phai su dung ham de truy cap doi tuong?
        + tang tinh bao mat cho code (viet public nameCourse: string === nameCourse: string: deu hieu la public het)
    - tao ra 2 ham getter va setter
    - getter ba setter se co cung ten voi nhau
    - getter se lay gia tri ra
    - setter se set gia tri moi vo ham
    - cac thuoc tinh phai co them _ truoc ten thuoc tinh
    vd: 
        private _nameCommander: string;
        constructor(_nameCommander: string) {
        this._nameCommander = _nameCommander;
    }

    /**
     * Getter nameCommander
     * @return {string}
     */
	public get nameCommander(): string {
		return this._nameCommander +" la gia tri tra ve khi truy cap qua getter";
	}

    /**
     * Setter nameCommander
     * @param {string} value
     */
	public set nameCommander(value: string) {
		this._nameCommander = value;
	}
