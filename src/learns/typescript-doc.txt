Ham (Function)  
    -Cac ham deu tra ve mot ket qua (Phu thuoc vao kieu du lieu ma minh khai bao) --Ngoai tru kieu void
    -Giong cac ngon ngu cung khac, ta phai chi dinh kieu du lieu tu dau
    -Cac kieu tra ve: string, number, array, any

**De khai bao mot gia tri doi ben trong typescript(tinh den thoi diem 5/2022):

        - Khai bao mot bien co gia tri object de nhan cac doi tuong

Tai sao phai can ham trong ts:
    - Giup su ly nhung thao tac nghiep vu chung ma khong can phai viet lai qua nhieu
    - Mot nhom cac dong code dc dat mot cai ten de de quan ly

Tai sao can ham co tham so:
    - Giup linh dong truyen tham so dau vao de xu ly cac bussiness
    - co kha nang thay doi cac gia tri trong Ham bang gia tri dat trong dau ngoac tron - cac gia tri nay goi la tham so
    - viet tong quat, viet 1 lan, su dung nhieu lan
    - Cu phap khai bao:     
        + Phai co kieu du lieu tra ve
        + Cac kieu du lieu nhan vao phai biet thuoc kieu nao
        vd:
            function functionName(x:dataType, y:dataType) : dataTypeReturn {
                // Code Xu Ly
            }
    vd: 
        function tinhTbc(x:number, y:number) : void {
            var tbc =(x + y) /2;
            console.log(`trung binh cong cua ${x} va ${y} la: ${tbc}`);
        }

**Anonymous function: 
    - La mot Function khong co ten
    - VD: 
        function(x:number, y:string) : string {

        }
    - Vi khong co ten nen phai khai bao mot bien de nhan gia tri cua func dong
    Vd:     
        var demo1 = function(x:number, y:string) : string {

        }

    Tai sao phai du dung Anonymous function: 
        - tiet kiem bo nho -- binh thuong phai co hai khong nho: mot la khong gian nho cho bien, hai la khong gian nho cho ham
        - It ton khong gian nho -- chi ton mot khong gian nho (lap trinh ham truyen thong phai ton toi hai khong gian nho)

    -- Mot cach khai bao khac cua Anonymous function: 
        /**
        * Mot cach khai bao khac cua Anonymous function 
        * tach khai bao tham so va kieu gia tri tra ve sang mot ben 
        * Con lai ben trong xu ly nhu cu, moi thao tac nhu cu
        */
        let g_demo_anony_func: (x: number, y: string) => string = function(x,y) {

            return `Chao ${y}, nam nay ban ${x} tuoi co phai khong ?`;       
        }

    -- Arrow Function trong ts:
        /**
        * Arrow Function trong ts
        */
        var h = (x: number, y: number) : number => {
            return x +y;
        }
